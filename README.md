# 🏠 Brazil Real Estate Market Analysis
This project analyzes 20,000+ property listings in Brazil to identify regional price trends and the impact of property size on valuation.
### ✅ Milestone 1: Data Wrangling Complete
- Merged multiple datasets using Pandas.
- Standardized currency from BRL to USD (3.19:1 ratio).
- Engineered geospatial features (Lat/Lon) from raw strings.
